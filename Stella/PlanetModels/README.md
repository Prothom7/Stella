# Planet USDZ Models

Place your real planet `.usdz` files in this folder.

## Expected default filename

The AR view currently loads:

- `earth.usdz`

## Recommended sources

- NASA 3D Resources: https://nasa3d.arc.nasa.gov/models
- Sketchfab (filter for downloadable USDZ when available): https://sketchfab.com

## Notes

- Keep filenames lowercase and simple, for example: `earth.usdz`, `mars.usdz`, `jupiter.usdz`.
- If a model is very large or very small, the app auto-scales it for AR placement.
- If `earth.usdz` is missing, the app falls back to a generated sphere so AR still works.
